## Using this as a template

If you're a student looking for a clean LaTeX resume template, feel free to use this as a starting point.

Open in [Overleaf](https://www.overleaf.com) (upload the `.tex` file and hit compile), or run locally with `pdflatex clement_bosquet_cv.tex` if you have TeX Live / MiKTeX installed.
